# Siddharth Bansal — EECS Portfolio

Personal portfolio website for EECS and Computer Engineering undergraduate applications.

**Live:** [sidbansal.me](https://sidbansal.me)

## Stack

Pure HTML, CSS, and vanilla JS — no frameworks, no dependencies.

## Features

- Scroll progress bar and scroll-in animations
- Project modal with specs for LYNX, FIRE, CANIS, and LYRA
- Responsive layout (mobile, tablet, desktop)
- Apple-inspired monochrome design
- Open Graph and Twitter Card meta tags

## Structure

```
index.html   # Everything — styles, content, and scripts in one file
```

## Sections

1. Hero
2. Bio
3. Education (IBDP + IGCSE)
4. Internships
5. Projects (featured + other)
6. Skills & Technologies
7. Awards & Competitions
8. Extracurriculars
9. Timeline
10. Contact

## Deployment

Hosted on GitHub Pages with a custom domain (`sidbansal.me`).  
Push `index.html` to the root of the repo and point the domain A records to GitHub's IPs.
